jindo.m.TextArea

Version : 1.4.0
Form Element의 Textarea를의 입력값의 변화를 감지하여 자동으로 높이값을 증가시키는 컴포넌트

Sample

Description

TextArea에 글자 입력시 자동으로 TextArea의 Height을 증가시킨다.
TextArea에 라운드 처리를 지원한다.
단, TextArea 적용시 마크업과 CSS가이드를 준수하여야 한다.

Notice

√ 지원 브라우저

  • 안드로이드 2.2 이상
  • iOS 3.0 이상

√ 주의사항

안드로이드
Form에 focus,blur될 때 focus,blur 이벤트가 2번 발생한다.
Textarea에서 스크롤은 컨텐츠가 Textarea 영역보다 크고, 입력모드인 경우에 가능하다.
갤럭시Note 4.0.3, 갤럭시S2 LTE 4.0.3에서는 실시간으로 TextArea의 크기가 변경되지 않는다.
blur가 발생한 이후 TextArea에 상태가 변경된다.
iOS3, iOS4
Textarea에서 스크롤은 투핑거 스크롤로 작동한다. 반면, iOS5는 원핑거 스크롤이 가능하다.
안드로이드 2.1, iOS3, iOS4
Textarea영역 내부에서는 touch관련 이벤트가 발생하지 않는다.

Initialization

HTML
CSS
필수 CSS 요소는 다음과 같다.
  • [prefix] + textarea-focus 클래스 : Textarea Focus시 적용 되는 클래스
  • [prefix] + textarea-disable 클래스 : 비활성화시 Textarea에 적용 되는 클래스
Javascript
첫번째 파라미터는 기준엘리먼트의 Id 혹은 HTMLElement로 설정 할 수 있다.

사용자 디자인 적용 하기

HTML
Initialization의 HTML구조와 동일
CSS
Javascript
Initialization의 Javascript구조와 동일


√ TextArea 의 디자인을 사용자의 요구 사항에 맞게 변경하기 위해서는 각 CSS에 알맞는 속성을 적용 해야 한다.

  • [prefix] + textarea 클래스
  • 기본 TextArea 클래스를 지정한다. 주로 배경색 그라데이션 지정과 border 스타일을 지정 한다.
  • [prefix] + textarea-focus 클래스
  • TextArea 포커스시 배경색이나 border 부분 스타일을 지정한다.
  • [prefix] + textarea-disable 클래스
  • 비활성화되어 선택이 불가능한 TextArea의 투명도, 커서 등의 스타일을 지정한다.

Template으로 디자인 적용하기

자주 사용되는 디자인의 템플릿이다.
이미지를 클릭하여 Template 페이지를 열고, Template 페이지 하단의 소스를 Copy&Paste하여 적용한다.
폰트, 너비, 색상 등 스타일 변경이 필요하다면 소스내 주석을 참고하여, 원하는 부분의 스타일을 수정하여 적용한다.


  • 기본형 TextArea

  • 라운드 처리가 된 디자인형 TextArea
