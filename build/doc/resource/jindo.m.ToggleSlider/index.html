jindo.m.ToggleSlider

Version : 1.4.0
클릭 또는 드래그로 슬라이더 자를 토글시켜 On/Off를 설정할 수 있는 컴포넌트

Sample

Description

ToggleSlider 컴포넌트는 on,off의 값을 클릭 또는 드래그를 통해 값을 설정할 수 있는 컴포넌트이다.
ToggleSlider도 아래 그림과 같이 thumb와 track 객체로 구성된다.

Notice

√ 지원 브라우저

  • 안드로이드 2.1 이상
  • iOS 3.0 이상

√ 주의사항

안드로이드2.1
드래그로 Slide되는 기능을 지원하지 않는다.
기본 touchmove 이벤트가 제대로 발생하지 않는다.
안드로이드2.1의 경우 손을 화면에서 떼는 순간 touchmove가 발생하는 브라우저 버그가 있다.
touchmove이벤트를 제대로 발생시키기 위해서는 touchstart에서 기본동작을 중지시켜야 한다.
iOS 3.0
라운드 처리를 지원하지 않는다.
iOS 3.0의 경우 border-radius을 지원하지 않는다.

Initialization

HTML
  • 라디오버튼 엘리번트는 선택적으로 삭제할 수 있다.
  • 라디오버튼 엘리번트를 통해 현재 토글 슬라이더의 설정값을 얻어올 수 있다.
  • 라디오버튼 엘리번트는 [prefix]+'radio'의 name을 가진다.
  • track영역의 엘리먼트는 [prefix]+'track'의 클래스명을 가진다.
  • thumb영역의 엘리먼트는 [prefix]+'thumb'의 클래스명을 가진다.
CSS
필수 CSS 요소는 다음과 같다.
  • [prefix] + track 클래스 : track 영역에 적용되는 클래스
  • [prefix] + thumb 클래스 : thumb 영역에 적용되는 클래스
  • [prefix] + aron 클래스 : [ON] 텍스트에 적용되는 클래스
  • [prefix] + aroff 클래스 : [OFF] 텍스트에 적용되는 클래스
Javascript

사용자 디자인 적용 하기

HTML
Initialization의 HTML구조와 동일
CSS
Javascript
Initialization의 Javascript구조와 동일


√ ToggleSlider 의 디자인을 사용자의 요구 사항에 맞게 변경하기 위해서는 각 CSS에 알맞는 속성을 적용 해야 한다.

  • [prefix] + track 클래스
  • track 영역의 크기, 둥글림정도, 배경색 등의 스타일을 지정한다.
  • width,height을 조정하여 크기를 지정할 수 있다.
  • [prefix] + thumb 클래스
  • thumb 영역의 배경색, 둥글림정도 등의 스타일을 지정한다.
  • thumb의 크기는 사용자 임으로 수정할 수 있지만, width값은 track의 1/2, height값은 track과 같은 값으로 설정할 것을 권장한다.
  • [prefix] + aron 클래스
  • [ON] 텍스트의 폰트 스타일을 지정한다.
  • [prefix] + aroff 클래스
  • [OFF] 텍스트의 폰트 스타일을 지정한다.

Template으로 디자인 적용하기

자주 사용되는 디자인의 템플릿이다.
이미지를 클릭하여 Template 페이지를 열고, Template 페이지 하단의 소스를 Copy&Paste하여 적용한다.
폰트, 너비, 색상 등 스타일 변경이 필요하다면 소스내 주석을 참고하여, 원하는 부분의 스타일을 수정하여 적용한다.
