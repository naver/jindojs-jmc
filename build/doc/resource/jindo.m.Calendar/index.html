jindo.m.Calendar

Version : 1.4.0
정년도/월의 달력을 지정한 엘리먼트에 표시하는 컴포넌트이다. 미리 지정해놓은 엘리먼트에 삽입되기 때문에 원하는 디자인과 마크업 구조를 적용할 수 있다.

Sample

Description

Calendar는 별도의 HTML 마크업으로 구성다.
Calendar가 나타나고, 사라질때 fade-in/fade-out 효과를 지원한다.


[컨텐츠 구조]

Calendar 컨트롤영역
[이전 버튼], [다음 버튼] 그리고 "년도/월 표시영역"으로 구성된다.
[이전 버튼]을 이용하여 이전 달로 이동할 수 있다.
[다음 버튼]을 이용하여 다음 달로 이동할 수 있다.
"년도/월 표시영역"은 현재 보여지고 있는 Calendar가 년/월 정보를 노출한다.

Calendar 날짜영역
Calendar의 날짜가 표시되는 영역이다.
HTML 템플릿이 필요한 영역이다.

[닫기버튼]
클릭하면 Calendar가 사라진다.
[prefix]+ 'btn-close' 클래스 명을 갖는다.
사용자의 필요에 따라 생략 가능하다.

Notice

√ 지원 브라우저

  • 안드로이드 2.1 이상
  • iOS 3.0 이상

  • oSelectDate (Object) : 선택된 날짜 Date 정보 ex){nYear:2011 , nMonth:1, nDate:1}
  Calendar가 사라질때 호출된다. oSelectDate 사용하여 필요 동작을 수행하면 된다.
selectDate Calendar에서 일자를 선택 할 경우 발생합니다.

beforeMoveDate Calendar에서 현재 날짜에서 이전 달(년도) / 다음 달(년도)로 이동하기전 호출된다.

  • oOldDate (Object) : 이동 하기전 날짜 Date 정보 ex){nYear:2011 , nMonth:1, nDate:1}
  • oMoveDate (Object) : 이동할 날짜 Date 정보 ex){nYear:2011 , nMonth:1, nDate:1}
  • stop (Function) : 수행시 moveDate 이벤트가 발생하지 않음
  Calendar에서 현재 날짜에서 이전 달(년도) / 다음 달(년도)로 이동하기전 호출 한다. oOldDate, oMoveDate, 사용하여 필요 동작을 수행하면 된다.
moveDate Calendar에서 현재 날짜에서 이전 달(년도) / 다음 달(년도)로 이동할 경우 발생한다.
  • sType (String) : 커스텀이벤트명
  • oOldDate (Object) : 이동 하기전 날짜 Date 정보 ex){nYear:2011 , nMonth:1, nDate:1}
  • oMoveDate (Object) : 이동할 날짜 Date 정보 ex){nYear:2011 , nMonth:1, nDate:1}
  Calendar에서 현재 날짜에서 이전 달(년도) / 다음 달(년도)로 이동한 후 호출 한다. oOldDate, oMoveDate, 사용하여 필요 동작을 수행하면 된다.
