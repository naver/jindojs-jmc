클릭 또는 드래그로 슬라이더 바를 이동시켜 값을 설정할 수 있는 컴포넌트

Sample

Description

Slider 컴포넌트는 어떤 특정한 영역 내의 값을 슬라이더 바(thumb)를 이동하여 값을 입력받을 때 사용하기 적합한 컴포넌트이다.
Slider는 아래 그림과 같이 track과 thumb,range 객체로 구성된다.

Notice

√ 지원 브라우저

  • 안드로이드 2.1 이상
  • iOS 3.0 이상

√ 주의사항

안드로이드2.1
드래그로 Slide되는 기능을 지원하지 않는다.
기본 touchmove 이벤트가 제대로 발생하지 않는다.
안드로이드2.1의 경우 손을 화면에서 떼는 순간 touchmove가 발생하는 브라우저 버그가 있다.
touchmove이벤트를 제대로 발생시키기 위해서는 touchstart에서 기본동작을 중지시켜야 한다.

Initialization

HTML
  • thumb영역의 엘리먼트는 [prefix]+'thumb'의 클래스명을 가진다.
  • range영역의 엘리먼트는 [prefix]+'range'의 클래스명을 가진다.
CSS
필수 CSS 요소는 다음과 같다.
  • [prefix] + wrapper 클래스 : track 영역에 적용되는 클래스
  • [prefix] + range 클래스 : range 영역에 적용되는 클래스
  • [prefix] + thumb 클래스 : thumb 영역에 적용되는 클래스
Javascript

사용자 디자인 적용 하기

HTML
Initialization의 HTML구조와 동일
CSS
Javascript
Initialization의 Javascript구조와 동일


√ Slider 의 디자인을 사용자의 요구 사항에 맞게 변경하기 위해서는 각 CSS에 알맞는 속성을 적용 해야 한다.

  • [prefix] + wrapper 클래스
  • track 영역의 크기, 둥글림정도, 배경색 등의 스타일을 지정한다.
  • 가로형일 경우 width, 세로형일 경우 height을 조정한다.
  • [prefix] + range 클래스
  • range 영역의 배경색, 둥글림정도 등의 스타일을 지정한다.
  • 가로형일 경우 width, 세로형일 경우 height을 조정한다.
  • [prefix] + thumb 클래스
  • thumb 영역의 배경색, 둥글림정도 등의 스타일을 지정한다.

Template으로 디자인 적용하기

자주 사용되는 디자인의 템플릿이다.
이미지를 클릭하여 Template 페이지를 열고, Template 페이지 하단의 소스를 Copy&Paste하여 적용한다.
폰트, 너비, 색상 등 스타일 변경이 필요하다면 소스내 주석을 참고하여, 원하는 부분의 스타일을 수정하여 적용한다.

Custom Event

Name Desc Custom Event Object Properties
beforeChange Thumb이 움직이기 직전에 발생한다.
  • sType (String) : 커스텀이벤트명
  • nValue (Number) : 이동하려는 Thumb의 위치의 계산된 슬라이더 값. 위치 값에 따라 소수점 발생함
  • nPosition (Number) : 이동하려는 Thumb의 위치의 전체 track대비의 퍼센트 값
  • nAdjustValue (Number) : nAdjustPosition값으로 계산된 슬라이드 값
  • nAdjustPosition (Number) : nPosition 반올림하여 정수로 계산된 값
  • stop (Function) : 수행시 슬라이더가 이동하지 않으며 change 이벤트가 발생하지 않는다.
  Thumb의 움직임의 값을 조정하고 싶을 때 함수 구현 예제
필요에 따라 정수가 아닌 소수 2째자리 까지 슬라이더 값을 허용하고 싶을때 beforeChange 이벤트를 에서 nAdjustValue 값을 바꿔주면 된다.
change Thumb이 움직인 이후에 발생한다.
  • sType (String) : 커스텀이벤트명
  • nValue (Number) : 이동하려는 Thumb의 위치의 계산된 슬라이더 값.위치 값에 따라 소수점 발생한다.
  • nPosition (Number) : 이동하려는 Thumb의 위치의 전체 track대비의 퍼센트 값
  • nAdjustValue (Number) : beforeChange에서 다시 설정한 슬라이더 값(변경된 값이 없으면 nValue의 반올림한 정수의 값)
  • nAdjustPosition (Number) : nAdjustValue값에 대한 퍼센트값
  • stop (Function) : 수행시 영향을 받는것은 없다.
  Thumb의 움직임인 이후에 실행될 함수 구현
touchStart Thumb에 손을 터치 했을 때 발생
  • nValue (Number) : 현재 Thumb의 위치의 계산된 슬라이더 값
  • nPosition (Number) : 현재 Thumb의 위치의 전체 track대비의 퍼센트 값
  touch되었을 경우의 슬라이더 값을 알 수 있다.
touchEnd Thumb에 손을 떼었을 때 발생 .
  • nValue (Number) : 현재 Thumb의 위치의 계산된 슬라이더 값
  • nPosition (Number) : 현재 Thumb의 위치의 전체 track대비의 퍼센트 값
  사용자가 슬라이더에 손을 떼었을 경우 값을 알 수 있다.
