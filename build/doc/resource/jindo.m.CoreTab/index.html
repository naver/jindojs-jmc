jindo.m.CoreTab

Version : 1.4.0
탭컴포넌트 상위 컴포넌트

Sample

Description

jindo.m.CoreTab 컴포넌트는 Tab형태의 컴포넌트의 상위 클래스로, 탭컴포넌트에서 사용하는 공통적인 부분을 담당하는 컴포넌트이다.

Notice

√ 지원 브라우저

  • 안드로이드 2.1 이상
  • iOS 3.0 이상

Option

Name Desc Default
sClassPrefix Tab컴포넌트 class의 접두어를 지정한다. "tc-"
nDefaultIndex 탭생성시 선택될 탭의 index 정보. 0
nPanelDuration 패널 슬라이드 효과 사용시 애니메이션 동작 시간
0일 경우, 패널 슬라이더 효과가 발생하지 않는다. 단위는 ms이다. 		0
nHeight 패널 슬라이드 효과 사용시 패널의 Height값을 지정한다. 0
nWidth 패널 슬라이드 효과 사용시 패널의 Widht값을 지정한다.
0일 경우, width는 [sClassPrefix]panel-cont의 width(padding값은 제외)로 지정된다.		 0
bActivateOnload 컴포넌트가 로딩 될때 활성화 시킬지 여부를 결정한다. false로 설정하는 경우에는 activate()를 호출하여 따로 활성화 시켜야 한다. true
