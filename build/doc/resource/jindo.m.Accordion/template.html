제목탭과 내용탭이 쌍으로 펼쳐지고 접혀지는 컴포넌트

Sample

Description

Accordion 컴포넌트는 header영역을 클릭시 해당 block을 확장(Expand)/축소(Collapse)하는 형태의 동작을 수행한다.

Accordion 컴포넌트는 sClassPrefix+"block"으로 정의된 엘리먼트의 overflow속성을 hidden으로 만든 후, 사이즈를 조절하여 애니메이션 효과를 내는데 이 효과는 옵션을 통해 다양하게 지정할 수 있다.

[컨텐츠 구조]

block 영역
header와 body를 감싸는 영역이다.

header 영역
각 block을 컨트롤 하는 handler 역할을 하는 엘리먼트이다.

body 영역
확장(Expand) / 축소(Collapse)되는 내용에 해당하는 엘리먼트이다.


√ 1개의 block은 1개의 header와 1개의 body로 구성되어야하며, Accordion에는 1개 이상의 block이 있어야한다.

Notice

√ 지원 브라우저

  • 안드로이드 2.1 이상
  • iOS 3.0 이상

Initialization

HTML
  • 기준엘리먼트의 자식노드로 block엘리먼트가 최소 1개 이상 포함되어야 한다.
       block 엘리먼트는 [prefix]block 클래스명을 가진다.
  • block 엘리먼트는 제목부분인 dt엘리먼트와 내용부분인 dd엘리먼트로 구성된다.
  • [prefix]는 sClassPrefix 옵션으로 지정가능하다.
       디폴트 값은 "accordion-"이다.

Option

Name Desc Default
sClassPrefix 초기 HTML/CSS구조에서 필요한 className 앞에 붙는 prefix를 정의 "accordion-"
sDirection Accordion이 펼쳐질 방향
  • vertical : 세로
  • horizontal : 가로
 "vertical"
nDefalutIndex 디폴트로 확장될 인덱스, 선언하지 않을시 확장하지 않음 -1
bUseToggle Header에 클릭발생시 Block의 확장(Expand)/축소(Collapse) 토글여부 false
sTransitionTimingFunction Block의 확장(Expand)/축소(Collapse) Effect효과
  • ease : 속도가 급가속되다가 급감속되는 효과 (거의 끝에서 급감속됨)
  • linear : 등속효과
  • ease-in : 속도가 점점 빨라지는 가속 효과
  • ease-out : 속도가 천천히 줄어드는 감속효과
  • ease-in-out : 속도가 천천히 가속되다가 천천히 감속되는 효과 (가속과 감속이 부드럽게 전환됨)
 "ease"
nDuration Block의 확장(Expand)/축소(Collapse) Effect처리 지속시간 500

  • nBeforeIndex (Number) : 기존에 확장(Expand)되어있는 Block의 인덱스
