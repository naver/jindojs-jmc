Form Element의 CheckBox를 모바일에 환경에 맞게 커스터마이징한 컴포넌트.

Sample

Description

CheckBox의 Unit영역을 터치하여 선택 할 수 있으며, 옵션에 따라 가로형, 세로형 UI를 제공한다.
jindo.m.CheckRadioCore 클래스를 상속 하여 생성 하기 때문에 해당 JS 소스를 선언해 주어야 한다.
단, CheckBox 적용시 마크업과 CSS가이드를 준수하여야 한다.



기능 설명
  • 일부 / 전체 활성화, 비활성화 기능
  • 일부 / 전체 선택, 선택 해제 기능
  • 선택 / 선택 해제시 스타일 적용
  • 활성 / 비활성시 스타일 적용

Notice

√ 지원 브라우저

  • 안드로이드 2.2 이상
  • iOS 3.0 이상

Initialization

HTML
컴포넌트는 CheckBox Unit 영역과 여러개의 Unit 영역을 감싸는 CheckBox Container 영역으로 구성된다.
  • Container 영역의 엘리먼트는 [prefix]+'checkbox-cont'의 클래스명을 가진다.
  • Unit 영역의 엘리먼트는 [prefix]+'checkbox-unit'의 클래스명을 가진다.
  • Unit 영역은 checkbox 타입의 Input 엘리먼트와 label 엘리먼트로 구성된다.
  • checkbox 타입의 Input 엘리먼트는 [prefix]+'checkbox-unit'의 클래스명을 가진다.
  • label 영역의 엘리먼트는 [prefix]+'checkbox-label'의 클래스명을 가진다.
  • 로딩 후 선택될 엘리먼트의 경우 Input 엘리먼트에 'checked' 속성을 추가한다.
CSS
필수 CSS 요소는 다음과 같다.
  • [prefix] + checkbox-on 클래스 : 선택시 CheckBox Unit 영역에 적용되는 클래스
  • [prefix] + checkbox-disable 클래스 : 비활성화시 CheckBox Unit 영역에 적용되는 클래스
Javascript
첫번째 파라미터는 기준엘리먼트의 Id 또는 HTMLElement로 설정 할 수 있다.

√ CheckBox 의 디자인을 사용자의 요구 사항에 맞게 변경하기 위해서는 각 CSS에 알맞는 속성을 적용 해야 한다.

  • [prefix] + checkbox-on 클래스
  • 클릭으로 선택된 Checkbox의 폰트, 배경색 등의 스타일을 지정한다.
  • [prefix] + checkbox-disable 클래스
  • 비활성화되어 선택이 불가능한 Checkbox의 투명도, 커서 등의 스타일을 지정한다.
  • [prefix] + checkbox-unit 클래스
  • 선택되기 전의 Checkbox에 기본으로 적용되는 스타일로, 폰트, 배경색, 너비, 높이등의 기본 스타일을 지정한다.
  • [prefix] + checkbox-unit:first-child 클래스
  • Checkbox 그룹에서 첫번째 Checkbox에 적용되는 스타일로, 주로 Radius 스타일을 지정하여 모서리 라운드 처리를 한다.
  • [prefix] + checkbox-unit:last-child 클래스
  • Checkbox 그룹에서 마지막번째 Checkbox에 적용되는 스타일로, 주로 Radius 스타일을 지정하여 모서리 라운드 처리를 한다.
  • [prefix] + checkbox-label 클래스
  • CheckBox의 설명을 표시하는 label 영역에 적용되는 스타일로, 폰트, 여백등의 스타일을 지정한다.

Template으로 디자인 적용하기

자주 사용되는 디자인의 템플릿이다.
이미지를 클릭하여 Template 페이지를 열고, Template 페이지 하단의 소스를 Copy&Paste하여 적용한다.
폰트, 너비, 색상 등 스타일 변경이 필요하다면 소스내 주석을 참고하여, 원하는 부분의 스타일을 수정하여 적용한다.


  • 세로 기본형 CheckBox

  • 라운드 처리가 된 세로 기본형 CheckBox

  • 라운드 처리가 된 세로 우측형 CheckBox
    (check버튼이 우측에 위치)

  • 이미지를 이용한 세로 기본형 CheckBox

  • 이미지를 이용한 세로 우측형 CheckBox
    (check버튼이 우측에 위치)

  • 가로 기본형 CheckBox

  • 라운드 처리가 된 가로 기본형 CheckBox

Name Desc Custom Event Object Properties
checked Check Box 선택시 발생.
  • sType (String) : 커스텀이벤트명
  • elCheckBoxUnit (Elment) : CheckBox Unit 엘리먼트
  • elCheckBox (Elment) : CheckBox 엘리먼트

unchecked Check Box 선택해제시 발생.
  • sType (String) : 커스텀이벤트명
  • elCheckBoxUnit (Elment) : CheckBox Unit 엘리먼트
  • elCheckBox (Elment) : CheckBox 엘리먼트
  CheckBox 선택해제시 elCheckBoxUnit 과 elCheckBox 사용하여 필요 동작을 수행한다.

  • sType (String) : 커스텀이벤트명
  • aCheckBoxUnitList (Array) : 활성화 되는 CheckBox Unit 엘리먼트 배열
  • aCheckBoxList (Array) : 활성화 되는 CheckBox 엘리먼트 배열
  비활성화되었을 때 CheckBox Unit에 배경색을 지정하는 예제
