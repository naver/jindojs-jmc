Scroll이 종료된 시점을 알려주는 컴포넌트

Sample

Description

브라우저에서 제공하는 Scroll 이벤트는 모바일 단말기, OS마다 다르게 발생한다.
ScrollEnd 컴포넌트는 이런 Scroll 이벤트를 정제하여, 단말기 화면에서 실제 Scroll이 완료된 시점에 "scrollEnd" 사용자 이벤트를 발생시켜주는 컴포넌트이다.

Notice

√ 지원 브라우저

  • 안드로이드 2.1 이상
  • iOS 3.0 이상

√ 주의사항

    모바일 OS, 제조사에 따라 scroll이벤트의 발생순서와 횟수가 다르다.
iOS
touchstart - touchmove - touchend - scroll 순으로 이벤트가 발생한다.
scroll 이벤트는 스크롤이 완료된 후에 발생한다.
안드로이드
touchstart - touchmove(n개) - scroll(0 ~ n개) - touchend - scroll(0 ~ n개) 순으로 발생한다.
touch이벤트는 touchstart -> touchmove -> touchmove -> ... -> touchend 순으로 발생한다.
Android 2.3갤럭시S, S2 는 touchmove와 scroll 이벤트가 한 번씩 발생한다.
Android 2.1에서는 touchend 시점에 touchmove, scroll이 한꺼번에 발생한다.
안드로이드 3.0, 4.0
스크롤이 존재할 경우, touchend 이벤트가 발생하지 않는다. 이를 방지하기 위해서는 touchstart나 touchmove에서 preventDefault()를 이용하여 브라우저 기본기능을 제거해야만 touchend 이벤트가 발생한다.
touchend가 발생하지만, 스크롤, 클릭과 같은 기본기능이 동작하지 않는다.
ScrollEnd 컴포넌트는 Android3.0, 4.0에서, 주소창 내 스크롤종료시점을 확인 할 수 없다.

Initialization

Javascript
ScrollEnd 객체생성시, document.body의 스크롤 이벤트를 확인하여, scroll 완료 시점을 알려준다.

Custom Event

Name Desc Custom Event Object Properties
scrollEnd 스크롤이 완료된 시점에 발생하는 이벤트
  • sType (String) : 커스텀이벤트명
  • nTop (Number) : 스크롤 상위 포지션
  • nLeft (Number) : 스크롤 좌측 포지션
  스크롤이 완료된 시점에 이벤트가 발생한다.
