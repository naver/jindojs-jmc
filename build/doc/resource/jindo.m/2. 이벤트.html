진도 모바일 컴포넌트는 특정동작시점에 원하는 동작을 수행시킬 수 있도록 다양한 커스텀 이벤트를 제공한다. 각 컴포넌트의 attach, detach 메소드를 통해, 커스텀 이벤트를 등록하고, 해제할수 있다.

관련 내용은 [jindo.m.Component]의 attach,detach 메소드 사용법을 참조하기 바란다.

종류 및 발생순서

  • 진도 모바일 컴포넌트에서는 특정동작 시점의 전과 후에 각각의 커스텀 이벤트를 제공한다.
  • 발생하는 이벤트 순서는 "beforeEvent 커스텀 이벤트 => 동작시점 => event 커스텀 이벤트" 순으로 발생한다.
  • 꼭! 전, 후의 커스텀 이벤트를 모두 제공하지는 않는다. 동작 전 커스텀 이벤트만 제공할수도 있고, 동작 후 커스텀 이벤트만 제공할수도 있다

Name Desc Example
beforeEvent 동작시점 발생 전에 발생하는 커스텀 이벤트 beforePos : 포지션 잡기 전에 발생
event 동작시점 발생 후에 발생하는 커스텀 이벤트 pos : 포지션 잡은 후에 발생
