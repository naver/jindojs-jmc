진행 대기 상태를 알려주는 로딩 컴포넌트

Sample

Description

장기 작업 처리 시 사용자에게 처리중임을 알리는 UI 컴포넌트이다.

[Loading 구조]

Loading 컴포넌트는 로딩이미지, 로딩텍스트, Foggy로 구분되어진다.

기능

  • 전체 또는 특정 영역에 로딩 표시를 할 수 있다
  • 전체 또는 특정 영역에 Foggy기능을 지원한다
  • 이미지와 텍스트를 지정할 수 있다

Notice

√ 지원 브라우저

  • 안드로이드 2.1 이상
  • iOS 3.0 이상

√ 주의사항

안드로이드
z-Index 상 맨 앞에 있는 레이어에서 발생한 이벤트가 투과되어 뒤에 존재하는 레이어에 이벤트가 발생하는 문제가 있다.
iOS3.0, Android2.1
iOS3.0, Android2.1 에서는 로딩바가 돌아가지 않고, 깜빡이기만 한다.
내부적으로 keyframe Animation을 사용하는데, iOS3.0과 Android2.1은 keyframe Animation을 완벽하게 지원하지 않는다.

Initialization

CSS
화면에 보여지는 로딩의 z-index 속성은 내부적으로 1,000 으로 설정된다.
Javascript
생성자의 첫 파라미터가 null일 경우, 전체 화면을 기준으로 Loading이 생성된다.
생성자의 첫 파라미터가 HTMLElement인 경우, 지정한 엘리먼트 기준으로 Loading이 생성된다.

Option

Name Desc Default
nWidth 로딩이미지의 가로크기 31
nHeight 로딩이미지의 세로크기 31
sDefaultForeground 로딩이미지의 색상 black
sDefaultBackground 로딩이미지의 배경색 transparent
sLoadingText 로딩텍스트 내용 (HTML)
null인 경우, 텍스트가 표기되지 않는다. 		로딩중입니다
bUseFoggy Foggy 사용여부 전체화면 기준일 경우 true
부분화면 기준일 경우 false
sFoggyColor Foggy 색상 grey
sFoggyOpacity Foggy 투명도 0.3
bActivateOnload 컴포넌트 로드시 activate 여부 true
