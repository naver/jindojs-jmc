jindo.m.CheckRadioCore

Version : 1.4.0
Form Element의 CheckBox / RadioButton 컴포넌트의 Core 클래스.

Description

CheckBox / RadioButton의 공통적인 기능을 처리 한다.
이벤트 처리 부분과 CheckBox / RadioButton 의 index 가져오기 등의 공통기능을 처리 한다.
