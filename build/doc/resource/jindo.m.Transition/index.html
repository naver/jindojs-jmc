jindo.m.Transition

Version : 1.4.0
jindo.m.Transition 컴포넌트는 엘리먼트의 css 스타일을 조정해 부드러운 움직임(변형)을 표현한다.

Sample

Description

Transition 컴포넌트는 엘리먼트의 css style의 변화를 주어 움직이는 효과를 주는 컴포넌트이다.
모바일에 최적화 하기위해 css3 Transtion을 이용한다.

Initialization

HTML
Javascript
"div1" 엘리먼트를 1초동안 opcaity를 0으로 설정하는 예제.

사용 예제

queue 메소드를 이용하여 실행할 transtion을 추가 할 수 있다.
queue 메소드의 첫번째 인자로는 대상 엘리먼트를 두번째 인자에는 실행 시간을 넣는다.
세번째 해시테이블에 css style 정보와 콜백정로를 입력하여 다양한 효과를 지정할 수 있다.
각 transtion별 엘리먼트와, 실행 시간, 애니메이션, 콜백 정보를 아래와 같은 방법으로 추가 한다.
style 속성 추가
기본 css의 style 속성을 변경 하려면 세번째 인자의 "htStyle"에 변경하려는 속성을 추가한다.
transform 속성 사용
css3의 transform 속성을 변경 하려면 세번째 인자의 "htTransform" 에 변경하려는 속성을 추가한다.
Transtion의 콜백 함수 사용예제
각 transition의 콜백함수를 지정하려면 세번째 인자의 "fCallback" 함수를 추가한다
Transtion의 콜 에 스타일을 추가하는 예제
각 transition의 콜백으로 CSS 스타일을 변경하려면 "fCallback"에 해시테이블로 추가 한다
형식은 기존의 css style과 3d css 속성을 추가하는 방법과 동일하다.

Option

Name Desc Default
sTransitionTimingFunction css Timeing function을 설정
  • ease : 속도가 급가속되다가 급감속되는 효과 (거의 끝에서 급감속됨)
  • linear : 등속효과
  • ease-in : 속도가 점점 빨라지는 가속 효과
  • ease-out : 속도가 천천히 줄어드는 감속효과
  • ease-in-out : 속도가 천천히 가속되다가 천천히 감속되는 효과 (가속과 감속이 부드럽게 전환됨)
 'ease-in-out'
bUseTimingFunction translate 혹은 translate3d 속성을 css3의 TimingFunction을 사용할지 여부.
false로 설정할 경우 자바스크립트의 setTimeout을 이용하여 애니메이션을 처리한다. 		모바일 단말기별로 다르게 설정된다.
상세내역은 [jindo.m]을 참조하기 바란다.

Custom Event

Name Desc Custom Event Object Properties
start Transition이 시작 될때 발생
  • sType (String) : 커스텀이벤트명
  • sTaskName (String) : 사용자가 설정한 sTaskName. 설정한 값이 없을 경우 null값 반환
  • element (HTMLElement) : Transition 대상 엘리먼트
  • nDuration (Number) : Transiton이 수행되는 시간
  • stop (Function) : 수행시 Transition이 실행되지 않는다. 전체 Transtion 실행도 멈춘다
 
end Transition이 끝날 때 발생
  • sType (String) : 커스텀이벤트명
  • sTaskName (String) : 사용자가 설정한 sTaskName. 설정한 값이 없을 경우 null값 반환
  • element (HTMLElement) : Transition 대상 엘리먼트
  • nDuration (Number) : Transiton이 수행되는 시간
  • stop (Function) : 수행시 영향을 받는것은 없다
 
stop Transition의 stop 메소드를 통해 중지하였을 때 발생
  • sType (String) : 커스텀이벤트명
  • sTaskName (String) : 사용자가 설정한 sTaskName. 설정한 값이 없을 경우 null값 반환
  • element (HTMLElement) : Transition 대상 엘리먼트
  • nDuration (Number) : Transiton이 수행되는 시간
  • stop (Function) : 수행시 Transition이 중지 되지 않고 그대로 실행된다.
