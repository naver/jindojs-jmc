Custom Event

Name Desc Custom Event Object Properties
beforeChange Thumb이 움직이기 직전에 발생
  • sType (String) : 커스텀이벤트명
  • nValue (Number) : 이동하려는 Thumb의 위치의 계산된 슬라이더 값.위치 값에 따라 소수점 발생함
  • nAdjustValue (Number) : nValue에서 반올림하여 정수로 계산된 값
  • stop (Function) : 수행시 슬라이더가 이동하지 않으며 change 이벤트가 발생하지 않습니다
  Thumb의 움직임의 값을 조정하고 싶을 때 함수 구현 예제
필요에 따라 정수가 아닌 소수 2째자리 까지 슬라이더 값을 허용하고 싶을때 beforeChange 이벤트를 에서 nAdjustValue 값을 바꿔주면 됩니다.
change Thumb이 움직인 이후에 발생
  • sType (String) : 커스텀이벤트명
  • nValue (Number) : 이동하려는 Thumb의 위치의 계산된 슬라이더 값.위치 값에 따라 소수점 발생함
  • nAdjustValue (Number) : beforeChange에서 다시 설정한 슬라이더 값(변경된 값이 없으면 nValue의 반올림한 정수의 값)
  • stop (Function) : 수행시 영향을 받는것은 없습니다
  Thumb의 움직임인 이후에 실행될 함수 구현
