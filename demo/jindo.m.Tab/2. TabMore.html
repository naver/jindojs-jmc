  • 탭 개수가 많을 경우, nCountOnList 옵션을 통해 더보기 탭을 구성할 수 있다.
    nCountOnList가 0 이 아니거나, 총 탭개수보다 작을 경우 더보기 탭으로 생성된다.
    만약, 모든 탭의 개수가 10개이고, nCountOnList가 3이면 3개의 탭과, 1개의 더보기 탭으로 구성된다.
  • '더보기' 문구 sMoreText 옵션을 통해 조절 할수 있다.

<더보기 탭에서는 더보기 부분을 구성할수 있는 추가 CSS 클래스를 제공한다.>

[sClassPrefix]more-tab : 더보기 탭을 class
[sClassPrefix]more-on : 더보기 탭을 선택했을 시 class
[sClassPrefix]arrow-down : 더보기 탭 옆의 화살표가 아래로 갈 경우 class
[sClassPrefix]arrow-up : 더보기 탭 옆의 화살표가 위로 갈 경우 class
[sClassPrefix]more-cont : 더보기 탭 선택시 나오는 목록박스 class
[sClassPrefix]more-li : 더보기 탭 선택시 나오는 목록박스의 아이템 class
[sClassPrefix]more-lia : 더보기 탭 선택시 나오는 목록박스의 아이템 속의 태그 class
