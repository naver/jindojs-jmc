※ 현재 Sample 보기에 적합하지 않은 브라우저를 사용중입니다. "크롬" 이나 "사파리" 브라우저를 이용해주시기 바랍니다.


new jindo.m.Dialog({ bUseEffect : false });

  • bUseEffect 옵션을 이용하여 인스턴스 생성
  • 따로 설정하지 않을 경우 { bUseEffect : true }로 설정됨
  • bUseEffect : true (default)

    Dialog를 열고, 닫을때 htEffectOption으로 설정한 Effect가 적용됨

  • bUseEffect : false

    Dialog를 열고, 닫을때 Effect 없음
