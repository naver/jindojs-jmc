Dialog.attach({ beforeShow : function() {...}, ...생략... });
  • 인스턴스에 customEvent를 attach하여 사용

    - 인스턴스가 Release될 때까지 이벤트가 유지됨

    Dialog.show({}, { confirm : function() {...}, ...생략... });

  • Dialog.show()시 customEvent를 parameter로 설정하여 사용

    - Dialog가 Hide되면 이벤트가 삭제됨

  • bAutoClose : true

    Dialog 이외의 영역 클릭시 자동으로 Dialog가 닫힘

  • bAutoClose : false (default)

    [확인], [취소], [닫기]버튼 클릭시에만 Dialog 닫힘
