※ 현재 Sample 보기에 적합하지 않은 브라우저를 사용중입니다. "크롬" 이나 "사파리" 브라우저를 이용해주시기 바랍니다.

  • 로딩이미지 크기 지정

    nWidth : 로딩모양의 width를 지정한다.

    nHeight : 로딩모양의 height를 지정한다.

  • 로딩이미지 색상 변경

    sDefaultForeground : 로딩이미지의 색상을 지정한다.

    sDefaultBackground : 로딩이미지의 배경색을 지정한다.

  • 로딩텍스트 변경

    sLoadingText : 로딩텍스트를 html으로 지정할 수 있다.

  • Foggy 변경

    bUseFoggy : 전체를 기준으로 로딩컴포넌트를 구성했을 경우, 기본으로 사용한다.

    sFoggyColor : Foggy의 배경색을 지정한다.

    nFoggyOpacity : Foggy의 투명도(opacity)를 지정한다.


<기본 옵션을 제외하고, 추가로 설정한 옵션은 아래와 같다.>

nWidth : 150,
nHeight : 150,
sDefaultForeground : red,
sDefaultBackground : transparent
sLoadingText : <strong><font color='red'>옵션이 변경된 로딩바입니다</font></strong>
bUseFoggy : true
sFoggyColor : green
nFoggyOpacity : 0.5
위 옵션 사용시, 150x150크기의 빨간색 로딩이미지가 '옵션이 변경된 로빙바입니다' 라는 빨간 텍스트와 함께 표시된다. 또한, Foggy는 투명도가 0.5인 그린색이다.
로딩컴포넌트가 보여지고 5초 후에 자동 사라집니다.
