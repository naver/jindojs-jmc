  • 스크롤을 터치 했을 때,beforeTouchStart, touchStart 이벤트가 발생한다.
  • 스크롤을 이동했을 때,beforeMove, move 이벤트가 발생한다.
  • 스크롤 터치를 끝냈을 때,beforeTouchEnd, touchEnd 이벤트가 발생한다.
  • 스크롤의 위치가 변경될 때,beforePosition, position 이벤트가 발생한다.
  • 스크롤 위치변경이 최종적으로 끝났을 때,afterScroll 이벤트가 발생한다.
  • pulldown, pullup이 발생할 때,pullDown, pullUp 이벤트가 발생한다.


<기본 옵션을 제외하고, 추가로 설정한 옵션은 아래와 같다.>

bUseHScroll : true,
bUseVScroll : false,
bUseMomentum : true,
nDeceleration : 0.0005,
nHeight : 110

<추가한 커스텀 이벤트는 다음과 같다>

beforeTouchStart : function(we) {
	addConsole(we.sType + " → " + "터시 전에 호출
	(nLeft:" + we.nLeft + ",nTop:" + we.nTop + ")");
},
touchStart : function(we) {
	addConsole(we.sType + " → " + "터시 후에 호출
	(nLeft:" + we.nLeft + ",nTop:" + we.nTop + ")");
},
beforeTouchEnd : function(we) {
	addConsole(we.sType + " → " + "터치 끝나기 전에 호출
	(nLeft:" + we.nLeft + ",nTop:" + we.nTop + ")");
},
touchEnd: function(we) {
	addConsole(we.sType + " → " + "터치 끝난 후에 호출
	(nLeft:" + we.nLeft + ",nTop:" + we.nTop + ")");
},
afterScroll : function(we) {
	addConsole(we.sType + " → " + "스크롤 위치변경이 최종적 완료
	(nLeft:" + we.nLeft + ",nTop:" + we.nTop + ")");
}

가로 가속이 가능한 스크롤에서 beforeStart, start, beforeEnd, end, scrollEnd의 커스텀 이벤트를 바인딩한 예제이다.

<참고>

beforeTouchMove, touchMove, beforePosition, position는 이동시 마다, 포지션 지정시 마다 빈번하게 발생하므로 이 예제에서는 바인딩하지 않았다
pullDown, pullup은 pull down/up update 기능이 있을시에만 발생하는 이벤트이기 때문에 바인딩하지 않았다
