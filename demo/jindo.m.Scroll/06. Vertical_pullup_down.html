더 보시려면 위로 올려주세요


  • 수평, 수직 스크롤은 bUseVScroll, bUseHScroll 옵션을 통해 조절가능하다.
  • 가속도는 bMomentum, nDeceleration 옵션을 통해 조절가능하다.
  • 스크롤바의 표기 여부는 bUseScrollbar 옵션을 통해 조절가능하다.
  • 스크롤 되는 고정영역의 크기는 nHeight, nWidth 옵션을 통해 조절가능하다.
  • Pull-down Update의 사용여부는 bUsePullDown 옵션을 통해 조절가능하다.
  • Pull-up Update의 사용여부는 bUsePullUp 옵션을 통해 조절가능하다.
  • Pull-down시 사용되는 문구는 fnPullDownIdle, fnPullDownBeforeUpdate, fnPullDownUpdating 옵션에 함수를 등록하여 조절가능하다.
  • Pull-Up시 사용되는 문구는 fnPullUpIdle, fnPullUpBeforeUpdate, fnPullUpUpdating 옵션에 함수를 등록하여 조절가능하다.
  • Pull-Down 사용여부를 setUsePullDown 메소드를 통해 동적으로 조절가능하다.
  • Pull-Up 사용여부를 setUsePullUp 메소드를 통해 동적으로 조절가능하다.

<기본 옵션을 제외하고, 추가로 설정한 옵션은 아래와 같다.>

bUseHScroll : false,
bUseVScroll : true,
bUseMomentum : true,
nDeceleration : 0.0005,
nHeight : 300,
bUsePullDown : true
bUsePullUp : true
fnPullDownIdle : function(wele) {
	wele.className("no_loading");
	wele.html("업데이트하시려면 아래로 내려주세요");
},
fnPullDownBeforeUpdate : function(wele) {
	wele.className("no_loading");
	wele.html("업데이트 합니다");
},
fnPullDownUpdating : function(wele) {
	wele.className("loading");
	wele.html("업데이트 중입니다...");
},
fnPullUpIdle : function(wele) {
	wele.className("no_loading");
	wele.html("더 보시려면 위로 올려주세요");
},
fnPullUpBeforeUpdate : function(wele) {
	wele.className("no_loading");
	wele.html("로드 합니다");
},
fnPullUpUpdating : function(wele) {
	wele.className("loading");
	wele.html("로드 중...");
},

위 옵션 사용시, 스크롤바가 있고, 위,아래로 가속(감속계수 0.0005) 가능한 pull-down/up 업데이트 스크롤러를 사용할 수 있다.

옵션 설정과 더불어, "pullDown", "pullUp" 이벤트를 구현하여 pullDown, pullUp시 스크롤내용을 변경한다
두 이벤트 모두 스크롤 내용을 변경한 후에는 꼭! refresh()를 호출하여 스크롤영역을 재계산하도록 한다
